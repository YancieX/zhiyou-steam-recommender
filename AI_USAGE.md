# AI使用说明

本文档记录本项目开发过程中AI工具的使用情况

## 使用工具

- **Deepseek**：前期准备工作、内容架构分析
- **Trae AI**：代码编写、UI开发、bug修复、文档生成
- **VS Code Copilot**：部分代码编写、注释生成

| 项目 | 内容 |

| 使用工具 | Deepseek |

## 总结

| 阶段 | AI使用程度 | 主要用途 |
|------|-----------|----------|
| 需求分析 | 中等 | 生成清单 |
| 可行性分析 | 中等 | 列分析框架 |
| 系统设计 | 中等 | 架构参考、结构建议 |
| 算法实现 | 较高 | 代码框架、原理讲解 |
| API集成 | 较高 | 主要靠AI查官方文档和调试 |
| 前端开发 | 较高 | 组件模板、样式初稿 |
| Bug修复 | 中等 | 分析错误、思路提供 |
| 文档编写 | 中等 | 格式框架、语言润色 |

**整体人工修改比例：约25-30%**

AI主要作为辅助工具，用于提高效率、减少重复劳动。核心的业务逻辑、软件设计、问题排查主要由自己完成。
