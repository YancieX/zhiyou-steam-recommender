# 技术架构文档

## 1. 架构设计

```
┌─────────────────────────────────────────────────────────┐
│                     前端应用 (SPA)                        │
│  React 18 + TypeScript + Vite + TailwindCSS             │
├─────────────────────────────────────────────────────────┤
│  页面层: Home | GameDetail | Wishlist | Profile | Admin  │
├─────────────────────────────────────────────────────────┤
│  组件层: GameCard | PriceAlert | RatingModal | NavBar    │
├─────────────────────────────────────────────────────────┤
│  状态层: Zustand Store (用户、游戏、愿望单状态)            │
├─────────────────────────────────────────────────────────┤
│  数据层: 模拟数据 (Mock Data) + 推荐算法                  │
└─────────────────────────────────────────────────────────┘
```

**说明**：本项目为纯前端演示系统，所有数据均使用浏览器本地存储（localStorage）模拟，不依赖后端服务器。

## 6. 推荐算法实现

### 6.1 用户相似度计算（余弦相似度）

```typescript
function cosineSimilarity(user1Ratings: number[], user2Ratings: number[]): number {
  const dotProduct = user1Ratings.reduce((sum, r, i) => sum + r * user2Ratings[i], 0);
  const norm1 = Math.sqrt(user1Ratings.reduce((sum, r) => sum + r * r, 0));
  const norm2 = Math.sqrt(user2Ratings.reduce((sum, r) => sum + r * r, 0));
  return norm1 && norm2 ? dotProduct / (norm1 * norm2) : 0;
}
```

### 6.2 混合推荐评分

```typescript
function hybridScore(cfScore: number, playtimeWeight: number): number {
  // cfScore: 协同过滤评分 (0-1)
  // playtimeWeight: 游玩时长权重 (0-1, 超过100小时归一化为1)
  return 0.6 * cfScore + 0.4 * playtimeWeight;
}
```
