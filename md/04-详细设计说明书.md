### 1.3 协同过滤算法伪代码

```
函数 协同过滤推荐(目标用户ID, 所有游戏, 所有评分, 所有用户):
    // 1. 构建用户-游戏评分矩阵
    为每个用户创建评分向量

    // 2. 计算目标用户与其他用户的相似度
    对每个其他用户:
        计算余弦相似度(目标用户向量, 其他用户向量)

    // 3. 找出最相似的K个用户
    按相似度排序，取前K个

    // 4. 计算候选游戏的预测评分
    对每个目标用户未评分的游戏:
        加权平均分 = Σ(相似用户评分 × 相似度) / Σ相似度

    // 5. 返回评分最高的N个游戏
    按预测评分排序，返回前N个
```

## 二、接口设计

### 2.1 内部接口

#### 2.1.1 UserStore

```typescript
// 登录
login(email: string, password: string): boolean

// 注册
register(username: string, email: string, password: string): boolean

// 退出登录
logout(): void

// 修改资料
updateProfile(updates: Partial<User>): void
```

#### 2.1.2 GameStore

```typescript
// 购买游戏
purchaseGame(userId: string, gameId: string, game?: Game): void

// 评分
rateGame(userId: string, gameId: string, score: number, playtimeHours: number): void

// 获取用户购买的游戏
getUserPurchasedGames(userId: string): Game[]

// 同步Steam价格
syncPricesWithSteam(): Promise<{success: number, failed: number, total: number}>
```

#### 2.1.3 WishlistStore

```typescript
// 添加愿望单
addToWishlist(userId: string, gameId: string, game?: Game): void

removeFromWishlist(userId: string, gameId: string): void

// 是否已在愿望单
isInWishlist(userId: string, gameId: string): boolean

// 检查降价
checkPriceChanges(): PriceAlert[]
```

### 2.2 外部API接口

#### 2.2.1 Steam Store 搜索接口

```
GET https://store.steampowered.com/api/storesearch
参数:
  - term: 搜索关键词
  - l: 语言：Chinesed
  - cc: 国家：cn
  - count: 返回数量
返回: 游戏搜索结果列表
```

#### 2.2.2 Steam 游戏详情接口

```
GET https://store.steampowered.com/api/appdetails
参数:
  - appids: 游戏ID
  - l: 语言
  - cc: 国家
返回: 游戏详细信息
```

#### 2.2.3 Steam 用户游戏库接口

```
GET https://api.steampowered.com/IPlayerService/GetOwnedGames/v1/
参数:
  - key: Steam API Key
  - steamid: 用户Steam ID
  - include_appinfo: 是否包含游戏信息
返回: 用户拥有的游戏列表和游玩时长
```
