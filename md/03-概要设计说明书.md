# 智荐优游 - 概要设计说明书

## 一、总体设计

### 1.1 系统架构

本项目采用纯前端SPA架构，前端应用运行于用户浏览器，无需后端服务器支持。

```
┌─────────────────────────────────────────────────┐
│                 用户浏览器                       │
│  ┌───────────────────────────────────────────┐  │
│  │                React 应用                 │  │
│  ├───────────────────────────────────────────┤  │
│  │  页面层                                   │  │
│  │  首页 | 详情 | 搜索 | 愿望单 | 个人中心     │  │
│  ├───────────────────────────────────────────┤  │
│  │  组件层                                   │  │
│  │  导航栏 | 游戏卡片 | 弹窗 | Toast          │  │
│  ├───────────────────────────────────────────┤  │
│  │  状态层                                   │  │
│  │  用户 | 游戏 | 愿望单 | 设置               │  │
│  ├───────────────────────────────────────────┤  │
│  │  工具层                                   │  │
│  │  推荐算法 | Steam API | 存储               │  │
│  └───────────────────────────────────────────┘  │
│                    │                           │
│  ┌─────────────────┴───────────────────────┐   │
│  │           localStorage                  │   │
│  │  用户 | 游戏 | 评分 | 愿望单 | 缓存       │   │
│  └─────────────────────────────────────────┘   │
└─────────────────────────────────────────────────┘
          │                │
          │                V
          │       ┌─────────────────┐
          └─────> │   Steam API     │
                  └─────────────────┘
```

### 1.2 技术选型

| 类别 | 技术 | 版本 | 选择理由 |
|------|------|------|----------|
| 前端框架 | React | 18.x | 生态成熟，组件化开发 |
| 开发语言 | TypeScript | 5.x | 类型安全，降低错误率 |
| 构建工具 | Vite | 6.x | 启动快速，热更新体验好 |
| 样式方案 | TailwindCSS | 3.x | 原子化CSS，提升开发效率 |
| 状态管理 | Zustand | 5.x | 轻量级，API简洁 |
| 路由管理 | React Router | 7.x | 官方推荐路由方案 |
| 图标库 | Lucide React | 最新 | 图标丰富，样式统一 |

### 1.3 第三方服务

| 服务 | 用途 |
|------|------|
| Steam Store API | 游戏搜索、详情、价格查询 |
| Steam Web API | 用户游戏库导入（需API Key） |
| corsproxy.io | 跨域代理服务 |
| images.weserv.nl | 图片代理服务 |

### 1.4 存储方案

本项目采用浏览器localStorage进行数据持久化，纯前端实现，无需后端支持。

- 优点：实现简单，无需服务器
- 缺点：容量有限，不支持多设备同步
